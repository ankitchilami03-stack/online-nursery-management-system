# Online Nursery Management System

## Project Description

The Online Nursery Management System is a web-based application that allows farmers and users to browse and purchase plants online. The system helps nursery owners manage plant inventory, orders, and deliveries efficiently. Users can explore different plant categories and place orders through an easy-to-use interface.

## Technologies Used

* HTML
* CSS
* JavaScript
* Bootstrap
* PHP
* MySQL

## Features

* User registration and login system
* Browse plants by categories such as Flowers, Fruits, Sugarcane, and Vegetables
* Add plants to cart and place orders
* Online payment integration
* Admin dashboard to manage plants and orders
* Delivery boy management and order assignment
* Order tracking and delivery status updates

## Modules

### Admin

* Manage plant categories and products
* View and manage customer orders
* Assign delivery boys to orders

### User (Farmer)

* Register and login
* Browse plants by category
* Add plants to cart and place orders
* Make online payments

### Delivery Boy

* View assigned orders
* Update delivery status

## Purpose

The main objective of this project is to provide an online platform where farmers can easily purchase plants from nurseries and nurseries can efficiently manage their plant sales and deliveries.

## How to Run the Project

1. Install XAMPP or any PHP server.
2. Import the database file into phpMyAdmin.
3. Place the project folder inside the `htdocs` directory.
4. Start Apache and MySQL in XAMPP.
5. Run the project in a browser using localhost.
